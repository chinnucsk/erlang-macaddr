Description

Cross platform MAC address determination. Works for:
* /sbin/ifconfig
* /bin/ifconfig
* ifconfig
* ipconfig /all 

   To return the first MAC address on the system:
     macaddr:address()
   

   To return an array of all MAC addresses:
      macaddr:address_list
